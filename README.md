# **Sports-Equipment-Store**

**Live Site**: [Sports Equipment Store](https://equipment-store-a8bbb.web.app/)

---

### **Overview:**

The **Sports-Equipment-Store** is a platform designed to showcase and manage a wide range of sports equipment. Users can view, add, and customize their sports equipment listings, with features like secure authentication, a dynamic theme switcher, and a responsive layout.

---

### **Key Features:**

1. **Categorized Sports Equipment**:  
   Easily navigate through various types of sports equipment with categorized views.

2. **Secure User Authentication**:  
   Users can register and log in securely to manage their equipment list and profile.

3. **Responsive Navbar**:  
   A fully responsive navbar with a mobile-friendly toggle menu and active link highlighting for better navigation.

4. **Dynamic Theme Switcher**:  
   Users can toggle between light and dark modes for a personalized browsing experience.

5. **Profile Customization**:  
   Users can personalize their profile with an avatar, a custom greeting, and a seamless logout option.

---

### **Technologies Used:**
- **Frontend**: React, React Router, Tailwind CSS
- **Authentication**: Firebase Authentication
- **Styling**: Tailwind CSS
- **Deployment**: Surge

### **Installation Instructions:**

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/jenifaa/sports-equipment-store.git
   cd sports-equipment-store
2. **Install Dependencies:** To install the necessary dependencies, run the following command in the project directory:

```bash

npm install
```
3. **Run the Project Locally**: To run the project on your local machine, use the following command:

```bash
npm run dev
```
4. **Set Up The Firebase Environment variable and .env file:**
5. **Set Up backend mongodb id,pass and token**
6.**Start backend:** 
```bash
nodemon index.js
